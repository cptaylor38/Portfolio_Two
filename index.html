Designer Delve

Visit site: Designer Delve

A simple gallery application that uses Sequelize and MySQL to store user Submitted art based on a few specified mediums. The app then generates an mdbootstrap modal with the submitted title, name of artist, email if supplied, facebook and linkedIn links if supplied, and a brief description of their art entry. The base structure of the html is using handlebars with dynamic script sources and an mvc workflow. The image uploads are using a cloudinary upload widget.
